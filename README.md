## PhD-Research-Recommendation-App
This project is a Flask-based web application that helps users find relevant PhD research topics by scraping multiple sources, including:  
    GitHub (for trending repositories related to the topic)
    arXiv (for academic papers) 
    CrossRef (for journal articles)
    OpenAlex (for research publications)

## Key Features:
Supports multiple research sources
Returns recommendations in JSON format
Generates a PDF report with results
Enables cross-origin access (CORS)
